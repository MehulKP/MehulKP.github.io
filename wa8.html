Denver Policing Data

Given the above visual, there was an overwhelming amount of arrests made in the first two days of November. This may be the result of the fact that Halloween occurs on October 31st so there may arrests being made in relation to crimes committed then.

Another cool finding is that the day that had the lowest number of arrests was Thanksgiving which was on November 22nd that year. This makes a lot of sense since presumably most people were staying at home/indoors with friends/family as opposed to breaking some laws.

Furthermore, it is worth mentioning the fact that arrest numbers quickly increase the subsequent day on Black Friday. Good ol' American consumerism.

That said, all of these changes in arrest numbers may not be the exlusive result of more/less crime, rather more/less police presence. Moving forward, it would be interesting to dig deeper into the data and attempt to find trends across days of the week and/or hours of the day via a heatmap to reflect these changes in crime and/or police presence.

Code Description:

  • Uses Pandas to read the policing data csv file in
  • Uses the groupby method and the Grouper function on the newly imported data group/filter the data by each row's "date" column value
  • These "date" groups are then aggregated to determine the total number of arrests made for each date. In other words, the sum number of arrests each day of November
  • A lil' bit of Dataframe manipulation so it was buttered up like a biscuit for some classic data viz action
